# Erigutemberg Meneses - Site Oficial

Site institucional oficial do Dr. Erigutemberg Meneses, advogado, economista, escritor e pesquisador independente, pioneiro no Neurodireito no Brasil e autor com mais de 30 obras publicadas.

## 🚀 Tecnologias utilizadas

- **HTML5** - Estrutura semântica e acessível
- **CSS3** - Design responsivo com Flexbox e Grid
- **JavaScript (Vanilla)** - Animações, menu mobile e formulário interativo
- **Font Awesome 6** - Ícones vetoriais
- **Google Fonts (Inter)** - Tipografia moderna
